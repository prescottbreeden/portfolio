Powder

Powder is a landing page for a mock heli-skiing company. Page is fully responsive and utilizes a number of CSS3 features and animations to provide a smooth user experience no matter what device they are browsing with.

HTML5, CSS3, Angular, JavaScript, SASS, AWS
